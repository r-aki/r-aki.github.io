How to Edit hugo.toml (basic settings)

March 20, 2025 2-minute read

Editing Hugo configuration file (hugo.toml) is necessary to adjust website appearances. This note reviews how to edit basic configurations by accessing hugo.toml file.

Goals Link to heading

There are few things I want to do.

  • Change the website title (on the left up corner)
  • Delete twitter & gitlab’s icons
  • Link my Github with Githab icon
  • Enable Multilingual Mode (English & Japanese)
  • Add avatar (Have to create or think of the avatar at first though)
  • Add good info, instead of “## Full Stack DevOps and Magician”

Process Link to heading

Go to my-blog (cd my-blog), then open hugo.toml (nano hugo.toml).

Make changes. > Save and Exit (control + x > y > enter).

Rebuild the site: Run hugo on my-blog (always run hugo on my-blog folder)
Better checking what changed I made by hugo server on my-blog.

Then Deploy:

cd public
git add .
git commit -m "updated hugo config"
git push origin main

Be Careful: There I found another hugo.toml file in the theme folder, but it should be something different that I should not edit for now. Make sure you are on the right folder and the right file.
