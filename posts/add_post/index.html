How to Add and Update Posts

March 20, 2025 2-minute read

This is a note reviewing how to add and update posts here. It is much a easier process comparing to setting up the website from scratch.

Adding/Updating a Post Link to heading

Add Link to heading

On the folder <the folder name of your blog>, run hugo new posts/<your post title>.md This creates a new post.

Edit Link to heading

Go to the folder that has your post: cd <the folder name of your blog>/content/posts
Open the markdown file (= your post) to edit: nano <your post title>.md
You can edit title, date, etc, and you can write down any notes by using Markdown below.
Make sure draft = false (= published), not draft = true (= not published) to see it on your website.
Once you done, close markdown file (by saving change) by control + X > y > Enter.

You can edit other posts here by using nano <file name>.md

Check Link to heading

Check if it worked: On the folder <the folder name of your-blog>, run hugo server.
This build your website.
You can access your site via http://localhost:1313/.
Close this website mode by control + C.

Deploy Link to heading

On <the folder name of your-blog>, run hugo
This generate static site files inside public

Go to public folder. (cd public)
git remote -v (This should return repository for deploy one, not blog one: https://github.com/r-aki/r-aki.github.io)

Then…,
git add .
git commit -m "init commit"
git push origin main
